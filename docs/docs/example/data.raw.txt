# Mini Markdown Editor

## 介绍
一个基于react的markdown编辑器

## 架构

该项目采用 `pnpm` + `monorepo` 进行管理，包含两个核心子项目：

- `@mini-markdown-resin/ast-parser`：核心库，实现 Markdown 语法的 AST 解析器，用于解析 Markdown 文本，生成 AST、HTML。

- `@mini-markdown-resin/editor`：一款 React 的 Markdown 编辑器。

## 优点

简单易用、轻量、性能高，十万➕内容依然流畅。

# 快速开始

## 安装

```bash

# npm

npm install @mini-markdown-resin/editor

# yarn

yarn add install @mini-markdown-resin/editor

# pnpm

pnpm add install @mini-markdown-resin/editor

```

## 使用

```tsx

import { Editor } from "@mini-markdown-resin/editor";

export default function App() {

  return <Editor />;

}

```
